Generate and rewrite text, or create images, right inside the editor. Pick a provider in the setup box below — the key, the editor, and the configuration all follow your choice. Requests go through the demo server, which holds your key.

Provider & API key

You're already viewing this through the demo server, so it's running. The only thing AI needs is the selected provider's key in server/.env — restart the server (cd server && npm start) after you add or change it.

# server/.env  —  GPT-4o (text) + gpt-image-1 (image)
GPT_API_KEY=your-openai-key# server/.env  —  Gemini (text only)
GEMINI_URL=https://…/gemini-2.0-flash:streamGenerateContent?alt=sse&key=…

AI Assistant

Click Text AI (✏️) in the toolbar to generate, rewrite, summarize, or translate — using the provider you picked above.

With GPT-4o you also get Image AI (🖼) to generate an image (uploaded to the server and inserted). Gemini is text-only here, so its image button is hidden.

Try it: select this paragraph, click Text AI, and ask for "Summarize this in one sentence."

Configuration

matches the selected provider 
// Load from CDN after synapeditor.min.js:
//   plugins/aiWriteSupporter/aiWriteSupporter.min.js + .min.css

var config = Object.assign({}, synapEditorLicense, synapEditorConfig, {
    'editor.menu.show': false,
    'editor.toolbar': ['undo', 'redo', '|', 'bold', 'italic', 'underline', '|', 'image', '|', 'openTextAI', 'openImageAI'],
    'editor.upload.image.api': '/uploadFile',
    'aiWriteSupporter.config': {
        text: { ai: 'gpt', url: '/requestGPT', requestBody: { model: 'gpt-4o' } },
        image: { url: '/requestGPTImage' }   // gpt-image-1
    }
});

var editor = new SynapEditor('synapEditor', config, initialHtml);
// Load from CDN after synapeditor.min.js:
