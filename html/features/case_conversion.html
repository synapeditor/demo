Case Conversion

Select the text below and click the Aa button in the toolbar to choose a conversion option.

Synap Editor is a Web-based WYSIWYG HTML editor.

  • UPPERCASE — SYNAP EDITOR IS A WEB-BASED WYSIWYG HTML EDITOR.
  • lowercase — synap editor is a web-based wysiwyg html editor.
  • Title Case — Synap Editor Is A Web-Based Wysiwyg Html Editor.
  • Toggle Case — sYNAP eDITOR IS A wEB-BASED wysiwyg html EDITOR. (inverts the case of every letter)

var config = Object.assign({}, synapEditorLicense, synapEditorConfig, {
    'editor.toolbar': [
        'new', '|',
        'conversion'
    ]
