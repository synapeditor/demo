Server Feature

OCR

Recognize the text and tables inside an image and insert them into the editor. Your image goes to the demo server, which calls Synap's OCR service and returns the result — your key stays on the server.

Image → text & tables Runs through the demo server

Your OCR key

You're already viewing this through the demo server, so it's running. OCR just needs your Synap OCR key in server/.env — restart the server (cd server && npm start) after you add or change it. Then click the OCR button in the toolbar.

# server/.env
OCR_API_KEY=your-synap-ocr-key

OCR

Click the OCR button in the toolbar, choose an image (a scan, screenshot, or photo of a document), and run recognition. The extracted text and tables are inserted right here.

Configuration

the editor sends the image to /ocr on the demo server 
// Load from CDN, after synapeditor.min.js:
//   plugins/ocr/ocr.min.js + ocr.min.css

var config = Object.assign({}, synapEditorLicense, synapEditorConfig, {
    'editor.menu.show': false,
    'editor.toolbar': ['undo', 'redo', '|', 'image', '|', 'ocr'],
    'editor.upload.image.api': '/uploadFile',
    // the OCR button uploads the image here; the demo server adds the key
    'ocr.config': { url: '/ocr' }
});

var editor = new SynapEditor('synapEditor', config, initialHtml);
