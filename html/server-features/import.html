Server Feature

Import

Open Office documents into the editor. Open sends the file to /importDoc for conversion; images inside the document are uploaded via /uploadFile so they render — so the upload API is required for imported images to appear.

server: converter-api api: /importDoc · /uploadFile

Import a Document

Click Open in the toolbar and choose a .docx, .xlsx, or .pptx file to load it here — including its images.

Requires the demo server and the converter-api container (HWP/HWPX is not supported by the converter).

Configuration

import + upload · external: SEDocModelParser 
// Requires externals/SEDocModelParser to be loaded (see <head>).
var config = Object.assign({}, synapEditorLicense, synapEditorConfig, {
    'editor.toolbar': ['new', 'open', '|', 'image'],
    'editor.import.api': '/importDoc',
    'editor.import.extensions': ['docx', 'xlsx', 'pptx', 'html', 'txt'],
    'editor.upload.image.api': '/uploadFile',
    'editor.upload.video.api': '/uploadFile',
    'editor.upload.file.api': '/uploadFile'
});

// Create editor instance
var editor = new SynapEditor('synapEditor', config, initialHtml);
