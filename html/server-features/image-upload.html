Image & File Upload

Insert images, videos, and attachments. The editor uploads each file to /uploadFile and embeds the returned URL.

Upload Media

Click the Image button in the toolbar and choose a file. It is uploaded to the demo server and inserted here.

Configuration

server: demo runner · upload api 
var config = Object.assign({}, synapEditorLicense, synapEditorConfig, {
    'editor.toolbar': ['new', 'image', 'video', 'file'],
    'editor.upload.image.api': '/uploadFile',
    'editor.upload.video.api': '/uploadFile',
    'editor.upload.file.api': '/uploadFile'
});

// Create editor instance
var editor = new SynapEditor('synapEditor', config, initialHtml);
