A license is required to run the demos

Without a valid license, the editor will not appear. Edit license.config.js at the project root and fill in your key before the demos work.

  1. Get a license — visit synapeditor.com, or email editorglobal@synapsoft.co.kr for inquiries.
  2. Set it up — edit license.config.js at the project root and fill in your company name and key. The key field is an array.
// license.config.js — at the project root
var synapEditorConfig = {
    'editor.license': {
        company: 'Your Company',
        key: ['YOUR-LICENSE-KEY']
    }
};
