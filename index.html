Licensing & how to run

license.config.js ships empty — get a license at synapeditor.com and add it before the editors will load. A free Evaluation license runs on http://localhost; a Production license runs on your registered domain.

  1. Get a license — an Evaluation / Issue license for local testing, or a Production license bound to your domain. Visit synapeditor.com or email support@synapeditor.co.kr.
  2. Set it up — edit license.config.js (the synapEditorLicense object) and fill in your company name and key. The key field is an array.
  3. Open over the right host — an Evaluation / Issue license is locked to the localhost hostname, so file://, 127.0.0.1, or any other IP make the editor fail silently. A Production license is bound to your registered domain — serve this demo from that same domain and it works there too.
// license.config.js — at the project root
var synapEditorLicense = {
    'editor.license': {
        company: 'Your Company',
        key: ['YOUR-LICENSE-KEY']
    },
    // Optional online load-check — leave empty to skip (default)
    'editor.license.load.api': { url: '', apiKey: '' }
};
